## Introduction

主要涉及到的数据结构:

* 字符串；
* 数组；
* 链表（单链表、双向链表）；
* 栈和队列；
* 二叉树；
* 哈希表；
* 图。

主要涉及到的算法:

* 排序算法：归并排序、快速排序、计数排序；
* 深度优先遍历、广度优先遍历；
* 动态规划；
* 贪心算法；
* 回溯。
