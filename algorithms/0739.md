# 739. 每日温度

## 题目地址

## 方法：单调栈

### 思路

最简单的方法时对每个温度进行遍历，找到后续温度中第一个大于该温度的温度。这样做的时间复杂度过高。



### 代码

```C++
class Solution {
public:
    vector<int> dailyTemperatures(vector<int>& T) {
        int size = T.size();
        vector<int> result(size, 0);
        stack<int> temp;
        for (int i = 0; i < size; ++i) {
            while (!temp.empty() && T[temp.top()] < T[i]) {
                int prev = temp.top();
                result[prev] = i - prev;
                temp.pop();
            }
            temp.push(i);
        }
        return result;
    }
};
```
